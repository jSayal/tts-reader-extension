TTS Reader Extension Test Page

How to use this test page:

  1. Make sure the TTS Reader extension is installed and enabled
  2. Configure your Google Cloud API key in the extension popup
  3. Select some text from the sample paragraphs below
  4. Either:
    • Right-click and select "Read selected text" from the context menu, or
    • Click the extension icon and press the "Speak" button in the popup

Sample Text for Testing

The Importance of Text-to-Speech Technology

Text-to-speech (TTS) technology converts written text into spoken voice output. This technology has become increasingly important in our digital world, providing accessibility for people with visual impairments or reading difficulties.

Modern TTS systems like Google Cloud Text-to-Speech use advanced neural networks to produce natural-sounding speech. These systems can recognize context, apply appropriate intonation, and even express emotions in some cases.

The applications of TTS are diverse and growing. From screen readers and audiobooks to virtual assistants and navigation systems, TTS technology is enhancing how we interact with digital content.

Try selecting this highlighted text and using the TTS Reader extension to hear it read aloud.

Extension Installation Verification

If the TTS Reader extension is properly installed, you should be able to:

  1. See the TTS Reader icon in your Chrome toolbar
  2. Right-click on selected text and see the "Read selected text" option in the context menu
  3. Hear the selected text being read aloud when using the extension

If any of these features are not working, please check the troubleshooting section in the README.md file.
